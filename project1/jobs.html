JD101 - Junior Software Developer

Salary Range

$65,000 - $85,000 AUD per year + superannuation

Overview

We're looking for a passionate and driven Junior Software Developer to join our growing development team. You'll be working closely with senior developers and project managers to build, test, and maintain high-quality software solutions. This is a great opportunity to grow your skills in a supportive, fast-paced environment that values learning and innovation.

Key Responsibilities

  1. Collaborate with the team to develop new features and maintain existing applications.
  2. Write clean, efficient, and well-documented code.
  3. Participate in code reviews and incorporate feedback to improve code quality.
  4. Assist in testing, debugging, and deployment processes.
  5. Work closely with UX/UI designers to implement user-friendly interfaces.
  6. Learn and adapt to new technologies and frameworks as needed.

Required Skills & Experience

  • Basic knowledge of one or more programming languages (e.g., Python, C#, JavaScript).
  • Understanding of software development principles and version control systems (preferably Git).
  • Strong problem-solving skills and a willingness to learn.
  • Ability to work both independently and in a team setting.
  • Good communication skills and attention to detail.

Preferred Qualifications

  • Familiarity with web development frameworks (e.g., React, Django, ASP.NET).
  • Experience working on personal or academic software projects.
  • Exposure to databases (e.g., SQL, MongoDB) and RESTful APIs.

What You'll Get

  • Mentorship from experienced developers.
  • Access to training materials and courses.
  • Opportunity to work on real-world projects from day one.
  • A relaxed and flexible working environment with a focus on growth.
